# Research Checklist

Use this checklist when researching a band or musician for the
`music-research` skill. Work through each category in order.

## 1. Core Facts

- [ ] Full name of the band/artist
- [ ] Date and place of formation / birth
- [ ] Genres
- [ ] Active years
- [ ] Record labels (past and present)
- [ ] Short description (2-3 sentences)

## 2. Members / Collaborators

For each member:

- [ ] Full name
- [ ] Role / instrument
- [ ] Active period with the band (joined -- left)
- [ ] Short bio (2-3 sentences on their contribution)
- [ ] Notable side projects or solo work
- [ ] At least 1 portrait image (prefer 2-3 across eras)

## 3. Lineup Changes

- [ ] Original formation lineup
- [ ] Every member departure and the reason (if public)
- [ ] Every new member addition
- [ ] Any reunions or temporary returns
- [ ] Dates for each change

## 4. Discography (Studio Albums)

For each album:

- [ ] Title
- [ ] Release year
- [ ] Short description (1-2 sentences on significance)
- [ ] Cover art image URL (public domain or official)

## 5. Major Events / Milestones

- [ ] Formation story
- [ ] Breakthrough moment
- [ ] Awards and accolades
- [ ] Landmark concerts or tours
- [ ] Controversies or public incidents
- [ ] Breakup / hiatus / reunion
- [ ] Deaths of members
- [ ] Any other culturally significant moments
- [ ] Dates and images for each event when available

## 6. YouTube Videos

Search for and collect URLs for:

- [ ] Top 5 most popular official music videos
- [ ] Top 3 live performance videos
- [ ] At least 2 videos from different eras of the artist's career
- [ ] Any historically significant performances (award shows, festivals, etc.)
- [ ] Verify each URL is accessible (not region-locked or removed)

Target: minimum 8 videos, ideally 12+.

## 7. Trivia / Fun Facts

- [ ] At least 3 notable facts about the artist
- [ ] Record-breaking achievements
- [ ] Unusual stories or little-known facts
- [ ] Cultural impact statistics

## 8. Visual Identity Cues

These feed into the design analysis that generates band-specific
visual direction concepts.

- [ ] Official band logo (image URL, note dominant colors and typography style)
- [ ] Dominant color palette across album covers and merchandise
- [ ] Genre mood adjectives (e.g. aggressive, ethereal, melancholic, raw)
- [ ] Era and cultural context (decade of peak activity, associated scene)
- [ ] Recurring visual motifs (skulls, nature, geometric, neon, lo-fi, etc.)
- [ ] Stage design and lighting cues (if notable)

## Recommended Sources

| Source | What to get |
|--------|-------------|
| **Wikipedia** | Core facts, history, discography, member info |
| **Wikimedia Commons** | Public-domain or CC-licensed images |
| **Official artist website** | Bio, discography, tour info |
| **YouTube (official channel)** | Official music videos, live performances |
| **AllMusic** | Discography details, genre classification |
| **Discogs** | Detailed release history |
| **MusicBrainz** | Structured metadata |
| **Setlist.fm** | Concert history |
| **Rock & Roll Hall of Fame** | Induction info, legacy |
| **Music press archives** | Rolling Stone, NME, Pitchfork for context |

## Quality Standards

- Prefer primary sources over fan wikis.
- Cross-reference dates between at least two sources.
- Use images that are publicly available (Wikimedia Commons preferred).
- For YouTube, prefer official artist channels over fan uploads.
- If a fact cannot be verified from two sources, mark it as unconfirmed
  in the dossier.
